Detailed Description

A Type.

Types can logically be classified as follows.

  • Physical. They can be immediately represented as bits in hardware.
  • Abstract. They can not implicitly be represented as bits in hardware without a more elaborate definition.
  • Primitive. These types contain no subtype.
  • Nested. These types contain some subtype.
